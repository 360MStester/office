# OFFICE在线测试系统 

**技术栈：**
SpringBoot、SpringData、MySQL、POI

**简介：**
基于B/S模式，做一个OFFIC在线测试系统，用OFFICE POI完成对word、excel、ppt的解析

**负责：**
和工作室组员合作完成，我负责整个项目的功能分析设计、数据库设计和服务器端的底层基本功能。
